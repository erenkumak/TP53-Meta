A differential gene expression heatmap can be drawn by the final gene list of the user. The samples can be clustered with the desired distance measure and linkage method. The interactive heatmap that shows the logFC value of each corresponding gene on the dendogram is generated by utilizing the [heatmaply](https://cran.r-project.org/web/packages/heatmaply/vignettes/heatmaply.html) package and can be exported as an html file.
<br><br>**Example**
<br>The previously filtered gene set is clustered based on Euclidean distance using complete linkage clustering for generating an expression heatmap.
<br><br>The clusters in which significantly up/downregulated genes such as *TUBA3E*, *SLPI*, *PGR* and *FUT7* belong can be examined.
